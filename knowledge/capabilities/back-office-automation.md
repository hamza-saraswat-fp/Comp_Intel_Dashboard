# State of the field

All three vendors ship only assistive, narrow AI back-office help today; none has shipped the deeper document/data automation (invoice OCR capture, autonomous AR/data entry) that defines a category leader, so there is no market_leading player here. Jobber leads on the single most substantive capability: its "Draft for me" / Automatic Draft Quotes genuinely constructs quotes (line items) from the request, past quotes, and templates, and can run autonomously on every new request — real generative estimating, not just summarizing text the user already typed. ServiceTitan has the broadest assistive surface (Pricebook Pro Price Insights regional benchmarking, Smart Start pricebook auto-population, invoice summary/email generators) but its heavier document automation — Automated Invoice Capture and Automated Proposal Templates — is explicitly "coming soon," so its GA footprint is shallow. Housecall Pro lags: its only concrete AI here, "Write it for me," merely drafts a summary from line items the user has already entered, and its proposal/pricebook tools are template-driven, not AI.
