# Assessment

Jobber's strongest AI area: a real AI voice/text receptionist that answers calls AND texts 24/7, qualifies and captures leads, books visits into the calendar, matches callers via caller-ID, transfers on keywords, and handles concurrent calls. Formally launched Aug 18, 2025 (200,000+ conversations handled) and sold as a priced add-on, it is a genuine voice agent rather than a chatbot — a real strength for an SMB tool, competitive with HCP. Calibrated to strong (not market_leading) since it is a single-product receptionist without ServiceTitan's full contact-center breadth.

# Detail

AI Receptionist: $99/mo add-on on Grow (and select plans); included in the Plus plan ($449/mo). Books jobs, captures requests, takes messages with task creation, transfers calls, caller-ID client matching, real-time owner notifications/monitoring with take-over, transcripts and recordings. Verified GA: the PR Newswire release (Aug 18, 2025) describes it as 'ready to work right out of the box' with 200,000+ conversations already handled; the feature page is publicly purchasable with Start Free Trial. The help-center article still labels it 'Beta,' but given the public launch, listed pricing, and purchasability, it is treated as shipped. Separately, Jobber AI Voice (included all plans) lets field users run 100+ in-app tasks hands-free by voice.
