# Assessment

Jobber AI genuinely constructs quotes — 'Draft for me' generates a quote (including suggested line items) in seconds from the request details, your saved templates, and your past quotes, and can run automatically on every new request via the Automatic Draft Quotes automation. This is real generative back-office automation for estimating, the most substantive single AI back-office capability among the three, hence strong — though it is narrow (quoting only, requires a few prior quotes to learn from, and has no invoice/receipt OCR or photo-to-quote).

# Detail

Verified via Jobber's product update and help-center content (re-confirmed by search; the help.getjobber.com and productupdates.getjobber.com pages block automated fetch but their content is corroborated): 'Draft for me' turns a request into a drafted quote using request details + saved templates + past quotes, suggesting line items based on previous jobs. It can be one-off or set to auto-run on every new request (Automatic Draft Quotes automation); enabling auto-draft requires at least 3 quotes sent via 'Draft for me' so the model has examples. High-value quote alerts notify when a high-value quote is auto-drafted. Limitation: no photo-to-quote and no invoice/receipt OCR or price-book ingestion, so document/data-entry automation is absent — narrower surface than ServiceTitan, but deeper on the one thing it does (it builds the quote rather than just summarizing typed-in items). Strong for SMB estimating; not market_leading because the document/data-automation half of this capability is missing.
